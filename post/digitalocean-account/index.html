Part 1—账户注册&建立云主机

最近貌似云主机很火，在知乎闲逛的时候经常可以看到这方面的讨论，BAT均推出了自己的云主机服务，除此之外还有微软、亚马逊、谷歌这些已经在国际上做云主机比较成熟的企业也针对国内市场提供服务了。在这些服务商中，我听到最多的要数Digital Ocean这家成立于2013年的公司，性价比超高，云主机包月价格竟然和国内的一些主机服务的价格一样，而且还有各式各样的优惠，最吸引我的就是要数GitHUb大礼包里边赠送的$100优惠券，算下来够用两年了，哈哈~

1.注册开通Paypal

由于Digital Ocean验证账户的方式是你需要使用信用卡或者Paypal往账户里边充5美元，考虑咱们学生目前没有信用卡，因此推荐使用Paypal充值。

Paypal就不用我多介绍了，钢铁侠在现实中的原型Iron Musk做的三件事之一，现在Paypal也和银联合作，只要银行卡上有银联标志都可以进行绑定账户，注册也比较简单，按照表单要求填就是了，点击这里可以直接进入Paypal中文主页。

接着就要选择Paypal账户类型了，如果你只需要使用Paypal付钱买东西而不是卖东西收钱的话，可以选择“个人”，开始填写基本信息，最后一项是你的银联卡号，Paypal会自动识别卡类型（虽然不知道为什么中国银行的借记卡识别不出来），按照要求填就好。

按照提示一步一步进行全部完成后，Paypal便注册开通了。

2.注册Digital Ocean账户

点击这里进入Digital Ocean账户注册页面，需要填写的内容其实很简单，按照页面要求进行填写就行。

在验证邮箱之后，需要往账户里充值才能激活账户，这里我们就可以使用之前申请的Paypal账户进行充值。

在注册的页面最下方有一个推广码填写，填写之后会免费送你10刀，大家可以填这个ae5f8c8a9e9f。如果你之前在github上验证了学生账户，github大礼包里边有一个100刀的优惠码，大家感兴趣的话可以尝试一下，链接在这里GitHub Student Developer Pack - GitHub Education

接下来是主机配置，包括选择主机名、服务器机房（建议旧金山）、系统（还可以选择一键集成的环境，我用的是CentOS 7）、主机配置（我选择的是最低的，其实也够啦~），这些选好后就可以尽情享用啦~

PuTTY基本介绍

PuTTY是windows环境下一款支持SSH远程登录的终端模拟器。好吧，如果你不知道什么是终端那就赶紧去看《鸟哥的Linux私房菜—基础篇》去吧！

PuTTY is a free implementation of Telnet and SSH for Windows and Unix platforms, along with an xterm terminal emulator

这里放出官方的下载页面链接：PuTTY Download Page

百度最让人讨厌的地方就是很多关键字搜索靠前的结果根本不是官方的内容，这就给了很多垃圾站很多机会，记得有次我点开下载竟然乱安装了一大堆东西，从此像这样需要了解官方的东西我就再也不用百度了，坑~大家可以用必应或者谷粉搜搜（国人自制谷歌日本镜像）

其实PuTTY就是一个文件而已（我喜欢这么简单的软件），直接点击putty.exe就可以下载了。这里大家再下载一个puttygen.exe，在教程的下一讲中将会用到。

软件的使用方法也很简单，只需要输入服务器ip，然后直接点open就好了，接下来会弹出一个模拟终端，按照提示输入用户名（root）和密码（发到你申请账户时填的邮箱中了），接下来就和教程里边讲的一样了。

Part 4—CentOS 7附加配置

此文作为自Digital Ocean社区教程的补充，教程链接地址：Additional Recommended Steps for New CentOS 7 Servers | DigitalOcean

在上一篇文章中，我们已经完成了CentOS的基本配置，包括建立user账户，配置账户权限以及锁定SSH。这篇教程将进行一些附加的但是建议的配置，就我的经验来看，这些配置其实是很重要的，尤其是防火墙以及swap的配置，一个是为了主机安全，另一个则是为了最大限度利用主机的性能，我最开始没有配置swap的时候配置LAMP的时候就遇到了内存不足的情况，Mariadb一度无法运行，在配置了swap之后问题得到了解决。

这一节的配置貌似没有什么难度特别大的地方，大家一步一步按照教程配置就行了。
