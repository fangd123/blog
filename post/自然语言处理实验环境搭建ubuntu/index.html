目前正在学习Coursera上的自然语言处理入门课程，本文章为Assignment 1的实验环境搭配，以供参考。

Ubuntu从0配置自然语言处理实验环境

实验指导上关于实验环境的配置写得比较简介，只列出了特定的版本要求，其他方面的配置一个字都没提，还得我找这些包的配置都弄了好半天（对，我是python新手） 最开始的课程要求里边也没写要多少python知识来着，不过还好，配置问题解决之后真正和Python代码相关的知识确实不多。

安装python

这个比较简单，直接输入sudo apt-get python就OK了，Ubuntu默认安装的版本是Python2.7，符合要求。

安装nltk

这个同样比较简单，我用的pip进行安装

安装scipy

根据官方文档的安装方法（简称一股脑安装法） http://www.scipy.org/install.html#individual-packages 直接输入

1

sudo apt-get install python-numpy python-scipy python-matplotlib ipython ipython-notebook python-pandas python-sympy python-nose

然后等着安装好，就啥事都没有了（至少在我的电脑上边是这样的）

安装numpy

难道你们没有注意到上边的一股脑安装法把numpy已经安装了么= =

安装sklearn

好吧，这个包把我坑了，实验指导上边写着一定要安装0.15.2的版本，但是我怎么都没找到这个版本的安装包 后来进入官网，发现人家包改名了，叫做scikit-learn，然后我就用pip install scikit-learn==0.15.2才安装成功的

运行程序

运行程序之后报错

UnicodeDecodeError: ‘ascii’ codec can’t decode byte 0xe5 in position 108: ordinal not in range(128)

一番搜索之后，将test.py文件中开头加入

1
2
3

import sys
reload(sys)
sys.setdefaultencoding('utf8')
