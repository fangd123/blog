如果按照层次来分的话，其实还需要把汇编加上去，这样的话从低到高就是汇编、C、C++、Java，这几种语言基本上可以完成任何事情，但是我觉得如果能够从底层了解语言的实现方法才能够称之为真正了解这门语言了，否则你不过是这门语言的使用者，而不能够驾驭这门语言，在这里，Java有些特殊，因为它属于解释型语言，存在一个JVM，如果想了解这个恐怕操作系统原理得学好。

语言本身没有好坏之分，但是作为学习计算机的，上述这几种语言应该做到都能够使用，并且大致了解原理，然后再精于一门语言就够了。这里边其实还涉及到了一些思想，网上资料很多，我就不多说了。

总之，它们都很重要，都应该学好。
